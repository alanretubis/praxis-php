Perfect for building small to medium web applications and APIs without the heavy baggage of larger frameworks.

   ```bash

   ```

2. Configure the Environment

Copy the example environment file to `.env`.

- For **macOS/Linux**:

  ```bash
  cp .env.example .env
  ```

- For **Windows**:

  ```bash
  copy .env.example .env
  ```
